Text formatting tags in HTML

Tags Description
<h1> -> <h6> This tags are used to represent headings in html form h1->h6 each heading is in different size.
< b > This tag is used for representing strong.
< i > This tag is used for italic.
< em > This tag defines emphasized text, with added semantic importance.
< mark > Thistag is used to present a part of text in one document as marked or highlighted for reference purposes.
< del > This tag is used to strike out the text
< u > This tag is used for underlying the text
< sub > This tag defines subscript texts. Subscript text is under the baseline of other symbols of the line and has smaller font.
< sup > tag defines superscript, that is set slightly above the normal line of type and is relatively smaller than the rest of the text.
< p > This tag is used for paragrahs in html.
< br > This tag is used for breaking the current line
< hr > This tag is used for a horizontal line
